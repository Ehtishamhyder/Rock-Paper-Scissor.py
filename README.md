# Rock Paper Scissor Game

This is a simple Rock Paper Scissor game developed in Python.

## Description
This program takes input from the user and randomly selects a choice for the Ehtisham.
Then it compares both choices and declares the winner.

## How to Run
1. Install Python
2. Download the file
3. Run the file using:

python filename.py

## Concepts Used
- Conditional Statements
- Random Module
- Basic Game Logic
